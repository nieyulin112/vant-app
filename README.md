# vant-app

基于vue-cli3.0+webpack 4+vant ui + less+ rem适配方案+axios封装，构建手机端模板脚手架

## 项目简单介绍

- vuecli3.0
- 多环境开发
- axios封装
- rem适配方案
- 生产环境cdn优化首屏加速
- babel低版本浏览器兼容
- 环境发布脚本
